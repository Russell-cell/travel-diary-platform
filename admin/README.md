# 项目技术栈

- React
- React-router
- Redux
- Antd
- Axios
- Mockjs

# 项目功能

- 实现用户登录
- 游记管理：展示所有用户发布的游记列表，可以对游记进行查看、审核、删除等操作。审核人员可以进行游记的审核通过和拒绝；管理员可以执行所有支持的操作 （ 通过、拒绝、删除）；拒绝需要提交拒绝理由。
- 两种角色：审核人员和管理员
- 支持通过不同的游记状态、发布时间、游记标题做筛选过滤
- 支持分页功能

# 项目启动

1. 进入项目目录 `cd travel_admin`
2. 安装依赖 `npm install`
3. 启动项目 `npm start`

在开发模式下运行应用。\
